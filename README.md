# Westcoast IT - Active Directory Project

## Project Overview
This project demonstrates the deployment and configuration of an Active Directory infrastructure on Azure for Westcoast IT, including domain services, DHCP, DNS, and organizational structure.

## Infrastructure Components
- **Domain**: WestCoast.local
- **Azure Resources**: Resource Groups, Virtual Networks, Virtual Machines
- **Services**: AD DS, DHCP, DNS, Print Services, Web Services

## Organizational Structure
```
WestCoast.local
├── Users
│   ├── IT
│   ├── HR
│   └── Sales
└── Computers
```
